# Utilities

Miscellaneous helper scripts used for performance and documentation workflows.

## Contents

- `benchmark_branches.R` - Compare script performance across Git branches.
- `dictate_dictionary.R` - Generate roxygen-style documentation for data frames.

## Usage

```r
source("utilities/benchmark_branches.R")
# Follow script prompts for branch selection and benchmarking.

source("utilities/dictate_dictionary.R")
dictate_dictionary()
```

## Notes

- This folder is for standalone helper tools that do not fit another category.
